## 3. 功能指南

### 3.1 模块一：用户管理

#### 3.1.1 查看个人信息

**功能说明**：查看和编辑当前登录用户的个人信息

**操作步骤**：
1. 点击右上角头像，选择【个人中心】
2. 在【基本信息】标签页查看个人信息
3. 点击【编辑】按钮修改信息
4. 修改完成后点击【保存】

**字段说明**：

| 字段 | 说明 | 是否可编辑 |
|------|------|------------|
| 用户名 | 登录账号 | 否 |
| 昵称 | 显示名称 | 是 |
| 手机号 | 绑定手机 | 是（需验证） |

**注意事项**：
- 手机号修改需要短信验证
- 修改密码需要输入原密码

## 4. 常见问题 (FAQ)

### Q1: 如何找回密码？

**A**: 
1. 在登录页面点击【忘记密码】
2. 输入注册手机号
3. 获取并输入短信验证码
4. 设置新密码

### Q2: 为什么收不到验证码？

**A**: 
- 手机号输入错误 → 检查手机号是否正确
- 短信被拦截 → 检查手机短信拦截设置
- 网络延迟 → 等待60秒后重新获取

### Q3: 系统支持哪些浏览器？

**A**: 推荐使用：Chrome 90+、Firefox 88+、Safari 14+

## 5. 故障排查

### 5.1 登录问题

#### 问题：无法登录，提示"账号或密码错误"

**排查步骤**：
1. **检查账号**：确认输入的是手机号而非用户名
2. **检查密码**：确认密码大小写正确
3. **检查账号状态**：确认账号未被禁用或锁定
4. **清除缓存**：清除浏览器缓存和Cookie

#### 问题：页面显示异常

**排查步骤**：
1. **检查浏览器版本**：确认使用支持的浏览器版本
2. **清除缓存**：按 Ctrl+Shift+Delete 清除缓存
3. **检查网络**：确认网络连接正常

### 5.2 功能使用问题

#### 问题：数据无法保存

**可能原因**：
- 网络连接中断
- 必填字段未填写
- 数据格式不正确
- 权限不足

**解决方法**：
1. 检查网络连接
2. 检查页面是否有红色提示（必填项）
3. 检查数据格式（如手机号、邮箱格式）
4. 确认当前账号有操作权限

4. 确认操作权限

## 性能问题

### 页面加载慢
1. 检查网络连接速度
2. 清除浏览器缓存
3. 尝试使用Chrome浏览器

### 操作卡顿
1. 刷新页面（F5）
2. 关闭并重新打开浏览器
3. 检查电脑内存使用情况
