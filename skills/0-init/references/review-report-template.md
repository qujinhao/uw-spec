# 通用评审报告模板

> 本模板为所有评审技能的共享报告模板。各评审技能的SKILL.md中不再内联报告模板，而是引用本文件。

## 评审发现记录

```markdown
## 评审发现

### 问题 1
**严重程度**: Critical / Major / Minor
**维度**: （按各评审技能定义）
**位置**: 文档名称/章节（或文件/行号）
**问题描述**: 具体问题描述
**修复建议**: 具体修复建议
**责任人**: @对应角色
```

## 量化通过标准

| 等级 | 条件 | 流转 |
|------|------|------|
| 通过 | >=95分，0个Critical | 进入下一阶段 |
| 不通过 | <95分，或存在Critical问题 | 返回当前阶段修改 |
