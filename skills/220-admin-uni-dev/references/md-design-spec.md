# 管理端移动端设计规范

> 面向管理人员和运营人员的移动端应用设计规范（移动审批、商户管理、运营后台小程序）。被 220-admin-uni-dev 引用。

## 色彩规范

管理端采用蓝色系，传达专业、高效、可信赖：

```css
--primary-color: #409EFF;      /* 主题色 - 操作按钮、导航栏 */
--success-color: #67C23A;      /* 成功色 - 审核通过、状态正常 */
--warning-color: #E6A23C;      /* 警告色 - 待处理、审核中 */
--danger-color: #F56C6C;       /* 危险色 - 拒绝、异常状态 */
--text-primary: #303133;       /* 主要文字 */
--text-regular: #606266;       /* 常规文字 */
--text-secondary: #909399;     /* 辅助文字 - 时间、状态说明 */
--border-color: #DCDFE6;       /* 边框颜色 */
--bg-color: #F5F7FA;           /* 页面背景 */
--card-bg: #FFFFFF;            /* 卡片背景 */
```

## 布局模式

### 导航栈模式（无 TabBar）

```
┌─────────────────────┐
│  ← 页面标题          │  ← navigationBar
├─────────────────────┤
│  [搜索] [筛选]       │  ← 搜索/筛选区（可选）
├─────────────────────┤
│  ┌───────────────┐  │
│  │ 数据项 1      │  │
│  │ 状态标签 时间  │  │
│  └───────────────┘  │
│  ┌───────────────┐  │
│  │ 数据项 2      │  │
│  └───────────────┘  │
│                     │
│        [+ FAB]      │  ← 悬浮新增按钮
└─────────────────────┘
```

### 页面结构

| 区域 | 说明 | 是否必须 |
|------|------|---------|
| 顶部导航栏 | 页面标题 + 返回按钮 | 是 |
| 搜索/筛选区 | 关键词搜索 + 状态筛选 | 列表页 |
| 数据列表 | 卡片式/行式列表 | 是 |
| 操作按钮 | 新增/编辑/删除（根据权限） | 视权限 |
| 底部悬浮 | 批量操作/提交（表单页） | 表单页 |

## 字体规范

| 级别 | 大小 | 字重 | 用途 |
|------|------|------|------|
| 页面标题 | 18px | 600 | navigationBar 标题 |
| 模块标题 | 16px | 600 | 区块标题 |
| 正文 | 14px | 400 | 常规内容、列表项 |
| 辅助文字 | 12px | 400 | 时间、状态说明 |

## 间距规范

| 类型 | 间距 |
|------|------|
| 基础间距 | 8px |
| 模块间距 | 16px/24px |
| 页面边距 | 16px |
| 卡片内边距 | 16px |
| 列表项间距 | 12px |

## 组件规范

### 列表项

```
┌─────────────────────────┐
│ 标题文字（16px, 600）     │
│ 辅助信息（12px, #909399） │
│ ┌────┐ ┌────┐            │
│ │标签│ │标签│   时间      │
│ └────┘ └────┘            │
└─────────────────────────┘
```

### 状态标签

| 状态 | 背景色 | 文字色 |
|------|--------|--------|
| 待处理 | #FFF3E0 | #E6A23C |
| 已通过 | #E8F5E9 | #67C23A |
| 已拒绝 | #FFEBEE | #F56C6C |
| 进行中 | #E3F2FD | #409EFF |

### 表单页

```
┌─────────────────────────┐
│ 表单字段 1               │
│ ┌─────────────────────┐ │
│ │ 输入内容             │ │
│ └─────────────────────┘ │
│ 表单字段 2               │
│ ┌─────────────────────┐ │
│ │ 选择内容  ▼          │ │
│ └─────────────────────┘ │
│                         │
│ ┌─────────────────────┐ │
│ │      提交            │ │  ← 主按钮（底部固定）
│ └─────────────────────┘ │
└─────────────────────────┘
```

## 平台适配

| 平台 | 设计要求 |
|------|---------|
| H5 | 响应式设计，适配各种屏幕 |
| Android | 原生体验，遵循 Material Design |
| iOS | iOS 原生体验，遵循 Human Interface |
| 微信小程序 | 遵循微信小程序设计规范 |

## 设计尺寸

- 设计基准: 375px (iPhone 6/7/8)
- 安全区域: 考虑刘海屏、圆角屏
- 点击区域: ≥ 44x44px

## 交互规范

| 交互 | 规范 |
|------|------|
| 下拉刷新 | 列表页支持下拉刷新数据 |
| 上拉加载 | 分页列表底部上拉加载更多 |
| 左滑操作 | 列表项左滑显示编辑/删除按钮 |
| 返回按钮 | 所有页面顶部导航栏显示返回按钮 |
| 操作反馈 | 操作后显示 Toast 提示 |
