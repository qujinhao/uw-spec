# 消费者端移动端设计规范

> 面向终端消费者的移动应用设计规范（电商小程序、内容App、社交应用）。被 220-guest-uni-dev 引用。

## 色彩规范

消费者端采用暖色系，传达活力、吸引力和亲和力：

```css
--primary-color: #E53935;      /* 主题色 - CTA按钮、选中态、高亮 */
--secondary-color: #FF7043;    /* 次主题色 - 促销标签、渐变辅助 */
--success-color: #43A047;      /* 成功色 - 购买成功、状态正常 */
--warning-color: #FFA726;      /* 警告色 - 库存紧张、提醒 */
--danger-color: #E53935;       /* 危险色 - 删除、错误 */
--text-primary: #212121;       /* 主要文字 */
--text-regular: #616161;       /* 常规文字 */
--text-secondary: #9E9E9E;     /* 辅助文字 - 时间、说明 */
--border-color: #EEEEEE;       /* 边框颜色 */
--bg-color: #F5F5F5;           /* 页面背景 */
--card-bg: #FFFFFF;            /* 卡片背景 */
```

## 布局模式

### TabBar 模式

```
┌─────────────────────┐
│  Logo    [搜索] [消息]│  ← 自定义顶部导航（部分页面）
├─────────────────────┤
│                     │
│    内容区域          │  ← 页面主体
│                     │
├─────────────────────┤
│  首页  分类  发现  我的│  ← TabBar（底部固定）
└─────────────────────┘
```

### 页面结构

| 区域 | 说明 | 是否必须 |
|------|------|---------|
| 顶部导航 | Logo/搜索/消息（首页）或返回标题（详情页） | 视页面 |
| Banner/轮播 | 顶部横幅广告、推荐内容 | 首页 |
| 内容列表 | 瀑布流、网格卡片、文章列表 | 是 |
| TabBar | 底部固定导航，Tab 数量由业务决定 | **固定配置**（项目必须配置） |

## 字体规范

| 级别 | 大小 | 字重 | 用途 |
|------|------|------|------|
| 大标题 | 20px | 700 | Banner 标题、活动标题 |
| 页面标题 | 18px | 600 | 页面标题 |
| 模块标题 | 16px | 600 | 区块标题、卡片标题 |
| 正文 | 14px | 400 | 常规内容、描述 |
| 辅助文字 | 12px | 400 | 时间、价格说明 |
| 价格 | 18px | 700 | 售价、促销价 |

## 间距规范

| 类型 | 间距 |
|------|------|
| 基础间距 | 4px/8px |
| 模块间距 | 12px/16px |
| 页面边距 | 12px/16px |
| 卡片内边距 | 12px |
| 网格间隙 | 8px |

## 组件规范

### 内容卡片

```
┌──────────────────┐
│                  │
│    [图片区域]     │  ← 封面图，圆角 8px
│                  │
│ 标题（2行截断）    │
│ 描述文字...       │
│ ★ 4.8  (1.2万)   │
│ ¥99  ~~¥199~~    │
└──────────────────┘
```

### TabBar 规范

| 属性 | 规范 |
|------|------|
| 图标尺寸 | 25x25px |
| 文字大小 | 10px |
| 未选中颜色 | #999999 |
| 选中颜色 | #E53935（主题色） |
| 高度 | 50px + 安全区 |
| 背景 | #FFFFFF，顶部 1px 边框 |

### 瀑布流布局

```
┌─────────┬─────────┐
│ 卡片 1  │ 卡片 2  │
│ (高)    │ (矮)    │
├─────────┤ 卡片 3  │
│ 卡片 4  │         │
├─────────┼─────────┤
│ 卡片 5  │ 卡片 6  │
└─────────┴─────────┘
```

## 平台适配

| 平台 | 设计要求 |
|------|---------|
| H5 | 响应式设计，适配各种屏幕 |
| Android | 原生体验，遵循 Material Design |
| iOS | iOS 原生体验，遵循 Human Interface |
| 微信小程序 | 遵循微信小程序设计规范，支持分享卡片 |

## 设计尺寸

- 设计基准: 375px (iPhone 6/7/8)
- 安全区域: 考虑刘海屏、圆角屏、底部指示器
- 点击区域: ≥ 44x44px

## 交互规范

| 交互 | 规范 |
|------|------|
| 下拉刷新 | 内容列表支持下拉刷新 |
| 上拉加载 | 底部上拉加载更多内容 |
| 左右滑动 | Banner 轮播、Tab 切换 |
| 长按操作 | 收藏、更多选项 |
| 分享 | 页面右上角分享按钮（微信小程序） |
| 返回顶部 | 长列表右下角悬浮返回顶部按钮 |
