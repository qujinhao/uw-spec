## PRD 详细模板

```markdown
# 产品需求文档 (PRD)

## 1. 文档信息
- 文档版本: v1.0
- 创建日期: YYYY-MM-DD
- 作者: XXX
- 评审状态: 待评审

## 2. 背景与目标
### 2.1 项目背景
描述项目的背景和业务痛点

### 2.2 项目目标
- 目标1: 具体可衡量的目标
- 目标2: 具体可衡量的目标

### 2.3 成功指标
- 指标1: 量化指标
- 指标2: 量化指标

## 3. 用户故事
### 3.1 目标用户
描述目标用户画像

### 3.2 用户故事
作为 [用户角色], 我希望 [功能], 以便 [价值]

## 4. 功能需求
### 4.1 功能模块1
#### 4.1.1 功能描述
详细描述功能

#### 4.1.2 业务流程
1. 步骤1
2. 步骤2
3. 步骤3

#### 4.1.3 页面原型
[链接或截图]

#### 4.1.4 验收标准
- [ ] 验收标准1
- [ ] 验收标准2

## 5. 非功能需求
### 5.1 性能需求
- 响应时间: < 200ms
- 并发用户: 10000+

### 5.2 安全需求
- 数据加密传输
- 敏感信息脱敏

## 6. 附录
### 6.1 术语表
| 术语 | 说明 |
|------|------|
| 术语1 | 说明1 |

### 6.2 参考文档
- 文档1
- 文档2
```
