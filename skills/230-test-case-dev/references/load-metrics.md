# 压测SLA指标模板

## 响应时间基线

### 按接口类型
| 接口类型 | P95 | P99 | 最大值 |
|---------|-----|-----|--------|
| 查询接口 | 100ms | 200ms | 500ms |
| 列表接口 | 200ms | 500ms | 1000ms |
| 提交接口 | 300ms | 800ms | 2000ms |
| 复杂计算 | 500ms | 1000ms | 3000ms |

## 容量指标

| 指标名称 | 目标值 | 说明 |
|----------|--------|------|
| 并发用户数 | 1000+ | 系统支持的同时在线用户数 |
| QPS | 1000+ | 每秒处理请求数 |
| TPS | 500+ | 每秒处理事务数 |
| 日活用户 | 10万+ | 支持的日活用户规模 |

## 资源指标

| 指标名称 | 目标值 | 说明 |
|----------|--------|------|
| CPU使用率 | < 70% | 服务器CPU平均使用率 |
| 内存使用率 | < 80% | 服务器内存使用率 |
| 磁盘I/O | < 80% | 磁盘I/O利用率 |
| 网络带宽 | < 70% | 网络带宽使用率 |

## 数据库指标

| 指标名称 | 目标值 | 说明 |
|----------|--------|------|
| 慢查询比例 | < 1% | 执行时间>1s的查询占比 |
| 连接池使用率 | < 80% | 数据库连接池使用率 |
| 缓存命中率 | > 90% | Redis缓存命中率 |

## 错误率指标

| 指标名称 | 目标值 | 说明 |
|----------|--------|------|
| HTTP错误率 | < 0.1% | 5xx错误占比 |
| 业务错误率 | < 1% | 业务逻辑错误占比 |
| 超时率 | < 0.5% | 请求超时占比 |

## 线程组配置参考

| 场景类型 | 线程数 | Ramp-up | 持续时间 | 目的 |
|---------|--------|---------|---------|------|
| 基准测试 | 1 | 1s | 循环1次 | 建立性能基线 |
| 负载测试 | 100~500 | 60s | 300s | 验证正常负载 |
| 压力测试 | 500~2000 | 120s | 600s | 找到性能极限 |
| 稳定性测试 | 200 | 30s | 3600s+ | 验证内存泄漏 |
