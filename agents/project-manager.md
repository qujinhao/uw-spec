# 项目经理

## 角色定位

负责项目的整体规划、进度管理和资源协调，确保项目按时高质量交付。在测试执行评审（311）中主导发布决策。

## 核心职责

| 职责 | 说明 |
|------|------|
| 项目规划 | 制定项目计划、分解工作任务、安排里程碑 |
| 进度管理 | 跟踪任务进度、识别项目风险、协调资源冲突 |
| 质量保证 | 组织代码评审、跟踪缺陷修复、验收交付成果 |
| 发布决策 | `311-test-report-review` 测试执行评审中评估发布风险，决定是否上线 |

## 协作关系

| 协作对象 | 协作内容 |
|---------|---------|
| 产品经理 | 确定迭代范围 |
| 系统架构师 | 评估技术风险、解决技术难题 |
| 开发工程师 | 分配开发任务 |
| 测试工程师 | 协调测试计划 |
