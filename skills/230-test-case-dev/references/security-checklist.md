# 安全测试检查清单

## SQL注入

| 检查项 | 测试方法 | 预期结果 |
|--------|---------|---------|
| 参数注入 | 输入 `' OR 1=1 --` | 请求被拦截或返回空结果 |
| 联合查询注入 | 输入 `UNION SELECT` | 请求被拦截 |
| 堆叠查询注入 | 输入 `; DROP TABLE` | 请求被拦截 |
| 盲注 | 基于时间/布尔的注入 | 响应时间无异常变化 |

## XSS（跨站脚本）

| 检查项 | 测试方法 | 预期结果 |
|--------|---------|---------|
| 反射型XSS | URL参数注入 `<script>alert(1)</script>` | 内容被转义或拦截 |
| 存储型XSS | 表单提交 `<img onerror=alert(1)>` | 存储内容被转义 |
| DOM型XSS | 通过DOM操作注入脚本 | 事件不被触发 |

## 越权访问

| 检查项 | 测试方法 | 预期结果 |
|--------|---------|---------|
| 水平越权 | 用A用户Token访问B用户数据 | 返回403或空数据 |
| 垂直越权 | 用低角色Token访问高角色接口 | 返回403 |
| IDOR | 修改URL中的资源ID | 无法访问他人资源 |
| API直接调用 | 绕过前端直接调用后端API | 权限校验仍然生效 |

## 敏感数据泄露

| 检查项 | 测试方法 | 预期结果 |
|--------|---------|---------|
| 响应脱敏 | 检查API返回的手机号、身份证 | 中间位用*号替代 |
| 密码明文 | 检查登录响应和数据库存储 | 密码使用bcrypt等哈希 |
| 错误信息 | 触发服务端异常 | 不返回堆栈信息和SQL |
| 日志泄露 | 检查日志输出 | 不包含Token和敏感字段 |

## CSRF（跨站请求伪造）

| 检查项 | 测试方法 | 预期结果 |
|--------|---------|---------|
| Token验证 | 移除CSRF Token提交请求 | 请求被拒绝 |
| Referer检查 | 修改请求Referer头 | 请求被拒绝 |
| SameSite Cookie | 检查Cookie配置 | SameSite=Strict/Lax |

## 认证授权

| 检查项 | 测试方法 | 预期结果 |
|--------|---------|---------|
| Token过期 | 使用过期Token访问 | 返回401 |
| Token伪造 | 修改Token内容 | 返回401 |
| 暴力破解 | 连续错误登录10次 | 账户锁定或验证码 |
| Session固定 | 检查登录后Session是否更新 | 登录后Session ID变化 |
