# 游客端Web设计规范

> 面向终端消费者的网站设计规范。被 220-guest-web-dev 引用。

## 色彩规范

### 品牌色

| 变量名 | 色值 | 用途 |
|--------|------|------|
| `--primary` | `#E53935` | 主品牌色，CTA 按钮、促销标签、购物车图标 |
| `--primary-hover` | `#C62828` | 主色悬停态 |
| `--primary-light` | `#FFEBEE` | 主色浅色背景，优惠券标签 |

### 中性色

| 变量名 | 色值 | 用途 |
|--------|------|------|
| `--foreground` | `#1A1A1A` | 主标题、重要文字 |
| `--foreground-muted` | `#666666` | 正文、商品描述 |
| `--foreground-subtle` | `#999999` | 辅助信息、时间戳 |
| `--background` | `#FFFFFF` | 页面背景 |
| `--background-muted` | `#F5F5F5` | 区块背景、商品卡片悬停 |
| `--border` | `#E5E5E5` | 边框、分割线 |
| `--border-strong` | `#D4D4D4` | 输入框边框 |

### 功能色

| 变量名 | 色值 | 用途 |
|--------|------|------|
| `--success` | `#22C55E` | 库存充足、支付成功 |
| `--warning` | `#F59E0B` | 库存紧张、限时提醒 |
| `--danger` | `#EF4444` | 错误提示、售罄 |
| `--info` | `#3B82F6` | 物流信息、帮助链接 |

### 暗黑模式

| 变量名 | 色值 | 用途 |
|--------|------|------|
| `--dark-background` | `#0A0A0A` | 深色背景 |
| `--dark-surface` | `#171717` | 卡片背景 |
| `--dark-foreground` | `#FAFAFA` | 深色模式文字 |

## 字体规范

| 级别 | 大小 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| Display | 36-48px | 700 | 1.1 | 首页大标题、活动 banner |
| H1 | 28-32px | 700 | 1.2 | 页面主标题 |
| H2 | 22-24px | 600 | 1.3 | 区块标题（"热销推荐"） |
| H3 | 18px | 600 | 1.4 | 商品名称、卡片标题 |
| Body | 15-16px | 400 | 1.6 | 商品描述、正文 |
| Caption | 13-14px | 400 | 1.5 | 价格、辅助信息 |
| Small | 12px | 400 | 1.4 | 标签、时间戳 |

### 价格展示规范

| 类型 | 样式 | 示例 |
|------|------|------|
| 现价 | 18px, 700, `--primary` | **$99.00** |

## 响应式断点

| 端 | 宽度 | 布局 |
|----|------|------|
| 移动端 | < 640px | 单列，商品 2 列网格 |
| 平板端 | 640px - 1023px | 商品 3 列网格，侧边抽屉导航 |
| 桌面端 | 1024px - 1279px | 商品 4 列网格 |
| 大屏 | 1280px+ | 商品 5-6 列网格，最大宽度 1440px |

## 阴影规范

| Token | 值 | 用途 |
|-------|-----|------|
| `--shadow-sm` | `0 1px 2px rgba(0,0,0,0.05)` | 输入框聚焦 |
| `--shadow-md` | `0 4px 12px rgba(0,0,0,0.08)` | 商品卡片悬停 |
| `--shadow-lg` | `0 8px 24px rgba(0,0,0,0.12)` | 弹窗、下拉菜单 |
| `--shadow-cart` | `0 -4px 20px rgba(0,0,0,0.1)` | 底部悬浮购物车 |

## 商品卡片规范

### 尺寸

| 场景 | 宽度 | 图片比例 |
|------|------|---------|
| 网格列表 | 自适应 | 3:4（竖图）或 1:1（方图） |
| 横向滚动 | 160-200px | 3:4 |
| 首页推荐 | 280-320px | 4:3 |

### 结构

```
┌─────────────────────────┐
│  [图片]  收藏按钮(右上)    │
│                         │
│  折扣标签(左上，可选)      │
├─────────────────────────┤
│  商品名称 (2行截断)       │
│  ★★★★☆ (4.8) 1000+ sold │
│  $99.00  ~~$129.00~~     │
│  [加入购物车] 或 变体选择   │
└─────────────────────────┘
```

### 交互状态

| 状态 | 样式 |
|------|------|
| 默认 | `--shadow-sm` |
| 悬停 | `--shadow-md`, 图片缩放 1.05 |
| 加载中 | 骨架屏，脉冲动画 |
| 售罄 | 图片灰度 60%，叠加 "Sold Out" 标签 |

## 暗黑模式适配

使用 Tailwind `dark:` 前缀或 CSS 变量切换：

```css
:root {
  --background: #FFFFFF;
  --foreground: #1A1A1A;
}

.dark {
  --background: #0A0A0A;
  --foreground: #FAFAFA;
}
```

Tailwind 配置启用 `darkMode: 'class'`，通过 `<html class="dark">` 切换。
